Executive Summary

The AutoX Facebook Video Uploader Bot is an AI-assisted automation system designed to simplify and optimize the process of uploading videos to multiple Facebook pages.

The bot automates posting, scheduling, and content management — saving time and ensuring consistency for creators, marketers, and businesses. Developed using Python (Flask) and a modern frontend, the system demonstrates how AI and APIs can be leveraged to make social media automation efficient and reliable.

Background & Problem

Social media marketing is vital, but manual video uploads to multiple pages are repetitive and time-consuming, leading to inconsistent posting and audience loss.

This inspired AutoX—an intelligent bot to handle Facebook uploads seamlessly.

Objectives & Scope

  • Automate video uploads and captions.
  • Enable post scheduling with custom intervals.
  • Allow multi-page management from one dashboard.
  • Enhance productivity through automation.

*Scope excludes ad management and analytics.

System Design & Methodology

Architecture Flow

  1. User provides Access Token & Page ID.
  2. Backend connects to Facebook Graph API.
  3. User uploads video and adds caption.
  4. Bot uploads or schedules the post.
  5. Logs are stored to prevent duplicates.

Working Process

  • Connect up to 5 Facebook pages.
  • Upload videos with titles/captions.
  • Set time intervals for auto-uploads.
  • Monitor logs for post status.
User → Flask Server → Facebook Graph API → Facebook Page

Implementation & Development

Challenges & Solutions

Challenge: Managing access tokens and authentication.

→ Solution: Used Long Lasting Facebook access tokens.

Challenge: Handling large video uploads without timeouts.

→ Solution: Implemented optimizations in file handling.

Challenge: Synchronizing scheduled uploads accurately.

→ Solution: Used precise delay logic for scheduling.
